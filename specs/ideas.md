# 향후 아이디어

스펙으로 전환되지 않은 기능 아이디어 목록. 우선순위 미정.

- **승인 대기 상태 표시**: PermissionRequest 이벤트 감지 시 에이전트 카드에 "AWAITING APPROVAL" 뱃지 표시. 승인 후 다시 active 상태로 전환
- **에이전트 상세 드로어**: 카드 클릭 시 오른쪽에 상세 패널 (이벤트 목록, 도구 사용 통계)
- **에이전트 그룹핑**: 프로젝트별로 에이전트 그룹화
- **대시보드 요약 뷰**: 전체 현황을 한눈에 보는 Overview 탭
- **실시간 이벤트 스트리밍**: 폴링 대신 WebSocket/SSE로 실시간 업데이트
- **차트 강화**: 도구별 사용량 시계열, 세션별 활동 히트맵
- **j/k 키보드 스크롤**: 목록 내 키보드 네비게이션
