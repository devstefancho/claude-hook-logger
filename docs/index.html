// Features

Everything you need to
understand Claude.

10 Hook Events

Captures every Claude Code lifecycle event — from SessionStart to SubagentStop, all logged in JSONL format.

Real-time Dashboard

Session timeline, tool usage charts, and interrupt detection — all visualized at localhost:7777.

AI Chat Analysis

Ask questions about your sessions via SSE-streaming Claude SDK chat with 6 structured MCP query tools.

CLI Tools

Terminal-based quick analysis with analyze-interrupts.sh and log-viewer.sh for instant insights.

MCP Integration

6 structured query tools so Claude can directly query its own hook event logs for deeper analysis.

macOS Desktop App

Native Electron app packaged as DMG. One-click launch — no terminal needed.

// Install

Get started in 3 commands.

Terminal
$ git clone https://github.com/devstefancho/claude-pulse.git
$ cd claude-pulse && pnpm install
$ ./install.sh
# ✓ hooks configured, claude-pulse CLI ready
 
$ claude-pulse --open
# ✓ dashboard running at localhost:7777
 
$ claude-pulse --doctor
# ✓ check installation health

That’s it. Claude Pulse hooks into Claude Code automatically. Use claude-pulse anywhere to manage the dashboard.

Requirements
macOS 12+ Node.js 18+ pnpm jq Claude Code
